My Work

NewsFinder App

Created a mobile application that returns any news article based on input from the user. Registration and login information is stored using the Firebase database. To narrow the news search, users will enter their preferences for country, query, language, category, domain, and page count. A short list of relevant news articles will appear with the option to "favorite" an article.

Run Game

Recreated the classic game "Run" in Unity. Users used WASD to move in the four directions and press space to switch the gravity. If a player runs into an obstacle, the game will reset automatically.

Space Shooter Game

Designed a space shooter game in Unity which includes a player ship and enemy ships. The player ship can shoot missiles using the keys 'A' and 'D'. The goal is to shoot the enemy ships which spawn in random waves of four from the top of the screen. When a missile collides with an enemy ship, an explosion occurs and the ship explodes into pieces. Game lasts until an enemy ship runs into the player ship.
